Solution

The solution here is to know about the existence of the Counter object.

n = 9
ar = [10, 20, 20, 10, 10, 30, 50, 10, 20]

def sockMerchant(n, ar):
    return sum([Counter(ar)[x] // 2 for x in Counter(ar)])

print(sockMerchant(n, ar))
